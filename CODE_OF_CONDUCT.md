# Code Of Conduct

## Our Standard

This project expects respectful, technically grounded collaboration. Critique the claim, the code, or the evidence directly. Do not make it personal.

## Unacceptable Behavior

- harassment or intimidation
- abusive language
- deliberate misrepresentation of evidence or scope
- pushing changes that hide adverse results

## Enforcement

The repository owner may remove content or restrict participation when behavior violates these standards.
